# menu-Тази задача реализира проста конзолна програма за управление на меню в ресторант. Създадени са класове Dish и Menu, които позволяват добавяне на ястия, показване на всички и наличните ястия, както и изчисляване на средната им цена. В главната програма се създава меню и се демонстрира работата на тези функционалности.
